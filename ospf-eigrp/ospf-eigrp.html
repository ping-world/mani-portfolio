What I’m building

I’m building a single GNS3 lab that combines an enterprise-style multi-area OSPF design with an EIGRP domain, then tying them together using dual redistribution so I can practice ENARSI-level routing and troubleshooting.

  • OSPF: multi-area, stub/totally-stub, NSSA, DR/BDR on multi-access, authentication, summarization, virtual-link scenario.
  • EIGRP: named-mode (or classic if required), stubs, unequal-cost load balancing (variance), summarization.
  • Redistribution: route-maps, tagging, feedback-loop prevention, primary/backup fusion points.
