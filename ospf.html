OSPF Multi-Area Lab

Topology

Area 0 backbone with Area 1 configured as stub. Three routers connected in linear topology: 

R1 ---- R2 ---- R3
Area0   Area0   Area1 (Stub)

Key Configuration

R1 & R2 (Area 0)

router ospf 1
 router-id 1.1.1.1
 network 10.0.0.0 0.0.0.255 area 0

R3 (Area 1 Stub)

router ospf 1
 router-id 3.3.3.3
 network 10.1.0.0 0.0.0.255 area 1
 area 1 stub

Verification

show ip ospf neighbor
show ip ospf database
show ip route ospf
show ip protocols

Expected Behavior

  • R3 receives default route from ABR
  • No external LSAs inside stub area
  • Full adjacency between Area 0 routers
