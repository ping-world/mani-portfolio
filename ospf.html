OSPF Multi-Area Lab

Topology

Multi-area OSPF network with Area 0 as the backbone (20.20.20.0/24 segment). Area 51 is dual-homed to the backbone via R2 and R4. Area 3 is a multi-access network off R3. Area 2 is a linear chain off R3. R1 provides external Internet connectivity. 

        Internet
           |
          (R1)
           |
  [   Area 0 Segment   ]
  [   20.20.20.0/24    ] -- (R3) -- (H1)
   /               \         |   \
 (R4)             (R2)       |    \
   \               /      Area 3  Area 2
    \             /      (Switch)  (R7)
     \           /        /    \     |
      \         /       (R9)  (R10) (R8)
       \       /
     [  Area 51  ]
        (R5)
         |
        (R6)

Key Configuration

R1 & R2 (Area 0)

router ospf 1
 router-id 1.1.1.1
 network 10.0.0.0 0.0.0.255 area 0

R3 (Area 1 Stub)

router ospf 1
 router-id 3.3.3.3
 network 10.1.0.0 0.0.0.255 area 1
 area 1 stub

Verification

show ip ospf neighbor
show ip ospf database
show ip route ospf
show ip protocols

Expected Behavior

  • R3 receives default route from ABR
  • No external LSAs inside stub area
  • Full adjacency between Area 0 routers
